# Xiaohongshu Keyword Design

Use this after the draft or content direction is mostly ready.

## Inputs

- title
- cover copy
- first 3 lines or first 3 seconds
- body copy or script
- intended audience and track

## Workflow

1. Identify core search word.
2. Add industry words.
3. Add audience or life-stage words.
4. Add pain/result words.
5. Add long-tail scenario words.
6. Limit to 10 final publishing keywords.
7. Check whether keywords appear naturally in title, cover, opening, body, and
   keyword field.

## Output

- final 10 keywords
- keyword type labels
- where each keyword should appear
- natural embedding suggestions
- warning if the keywords do not match the content
